# SPEC: v3.7 折叠改为默认全展开

## 问题
v3.5 的折叠功能：生成完成后默认只展开第一个 h2 区块，其余折叠。用户反馈想看全文。

## 方案
保留折叠交互能力（点击标题可折叠/展开），但**默认不折叠任何区块**。生成完成直接显示全文。

## 改动

**一个文件，删 4 行：**

`static/index.html:962-966` — 删掉「第一个展开，其余折叠」逻辑：

```js
// 删掉这段
    // First section expanded, rest collapsed
    if (i > 0) {
      h2.classList.add('collapsed');
      body.classList.add('hidden');
    }
```

标题的 ▶/▼ 箭头和点击折叠功能全部保留，只是初始状态改为全部展开。
